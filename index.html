What I Built

Designed and implemented a complete DevOps pipeline for a React + TypeScript financial web application. The system runs on an on-premise Kubernetes cluster hosted on Proxmox VMs, with full automation from code commit to live deployment ‚Äî achieving zero-downtime rolling updates.

Multi-Stage Docker Build
Node.js builder stage compiles React app, Nginx runner stage serves static assets ‚Äî minimal final image size.
Jenkins Declarative Pipeline
4-stage pipeline: Clone ‚Üí Build & Push ‚Üí GitOps Update ‚Üí ArgoCD Sync. Fully automated on git push.
Harbor Private Registry
Secure image storage with robot account authentication, Kubernetes pull secrets and versioned image tags.
GitOps with ArgoCD
Kubernetes manifests stored in Git. ArgoCD auto-syncs on manifest changes ‚Äî full audit trail of every deployment.
Zero Downtime Deployments
Rolling update strategy with maxUnavailable: 0 ‚Äî new pod healthy before old pod terminates.
Kubernetes Ingress + NodePort
Nginx Ingress Controller routes external traffic. NodePort exposes app across office network on Proxmox infrastructure.
